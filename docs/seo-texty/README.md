## Struktura každého textu

Každý text obsahuje:

- **Title** (max. cca 60 znaků)
- **Meta description** (cca 140–160 znaků)
- **H1 nadpis**
- **Úvodní text** (50–70 slov)
- **Minimálně 3 H2 sekce**, typicky:
  - Jak vybrat [typ produktu]
  - Co zvládne / jaký výkon očekávat
  - Typické konfigurace a varianty
  - Pro koho je kategorie ideální
  - Závěrečná CTA sekce
- **Délka:** 600–900 slov na kategorii
